# JARVIS - Personal Voice Assistant

Jarvis is a simple Python-based voice assistant inspired by AI systems like Alexa and Iron Man’s Jarvis. It can perform basic tasks like opening websites, giving time updates, and responding to voice commands.

---

## FEATURES

*  Voice command recognition
*  Open websites like YouTube & Google
*  Tell current time
*  Basic command processing
*  Lightweight and beginner-friendly

---

## TECH STACK

* Python
* SpeechRecognition
* Webbrowser
* OS Module

## INSTALLATION

1. Clone the repository:

```
git clone https://github.com/your-username/jarvis.git
```

2. Navigate to the folder:

```
cd jarvis
```

3. Install dependencies:

```
pip install -r requirements.txt
```

---

## USAGE

Run the program:

```
python main.py
```

Speak commands like:

* "Open YouTube"
* "Open Google"
* "What is the time?"
